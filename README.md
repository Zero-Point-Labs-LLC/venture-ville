# 🌆 Venture Ville

**A competitive startup board game of asymmetric engines and strategic tempo.**

## Project Overview

Venture Ville is a board game where players run competing tech startups, building capabilities, shipping products, and scaling platforms to achieve dominance through Users and Money.

**Status:** Core design phase — ready for turn loop development

## Core Mechanics

- **Roadmap Trees** — Unique company progression paths
- **Resource Economy** — Engineering, Design, Product, Marketing, Hiring
- **Talent System** — Partner → Pod → Team progression
- **Progress Cards** — Tactical event decks (5 disciplines)
- **Advancement Tracks** — Product & Engineering hierarchies
- **Funding Timeline** — Seed → Series A → Series B → Series C → IPO
- **Market Events** — Time pressure & volatility (7-roll replacement)

## Victory Conditions

Win through combination of:
- 👥 **Users** — market positioning & influence
- 💰 **Money** — scaling power & late-game dominance

## Design Philosophy

**Key Principles:**
- Asymmetric engines with timing parity
- Strategic divergence → convergence tension → platform control
- Shipping increases economic slope
- Shared archetypes, unique trees
- Runway pressure creates macro pacing
